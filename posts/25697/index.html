// 计算出生到现在的天数
       let day1 = new Date();  // 定义当前的时间
       let day2 = new Date("2001/06/08");  // 定义出生日期

       // 获取时间戳，毫秒
       let time1 = day1.getTime();
       let time2 = day2.getTime();

2.5 JSON 对象

什么是 JSON ?

JSON(JavaScript Object Notation, JS 对象简谱) 是一种==轻量级的数据交换格式==，类似Xml。简洁和清晰的层次结构使得 JSON 成为理想的==数据交换语言==。 易于人阅读和编写，同时也易于机器解析和生成，并==有效地提升网络传输效率==。

  • JSON 指的是 JavaScript 对象表示法（JavaScript Object Notation）
  • JSON 独立于语言：JSON 使用 Javascript语法来描述数据对象，但是 JSON 仍然独立于语言和平台。JSON 解析器和 JSON 库支持许多不同的编程语言。 目前非常多的动态（PHP，JSP，.NET）编程语言都支持JSON。
  • JSON 具有自我描述性，更易理解

格式：

  • 对象用{}
  • 数组用 []
  • 所有的键值对，都用 key：value

JSON 和 JS对象的转换

 // 普通的对象
        let user = {
            name: "夏一北",
            age: 20,
            sex: "男"
        }

        // 转换为 json 对象
        let JsonUser = JSON.stringify(user);
        console.log(JsonUser);  // 返回值是一个字符串，{"name":"夏一			  北","age":20,"sex":"男"}
        console.log(user);   // 返回值是一个对象，{"name":"夏一北","age":20,"sex":"男"}

// 通过字符串解析成一个对象
let JsonUser = JSON.parse('{"name":"夏一北","age":20,"sex":"男"}');
console.log(JsonUser);

JSON 和 JS 对象的区别

// js
var obj1 = {name: "jack", age: 18};
// JSON
var obj2 = '{'name': 'jack', 'age': 18}';

navigator

navigator对象表示浏览器的信息

  • navigator.appName：浏览器名称；
  • navigator.appVersion：浏览器版本；
  • navigator.language：浏览器设置的语言；
  • navigator.platform：操作系统类型；
  • navigator.userAgent：浏览器设定的User-Agent字符串。

screen

screen对象表示屏幕的信息，常用的属性有：

  • screen.width：屏幕宽度，以像素为单位；
  • screen.height：屏幕高度，以像素为单位；
  • screen.colorDepth：返回颜色位数，如8、16、24。

操作 DOM（重点）

DOM ： 文档对象模型

DOM 是一个树形结构，改变HTML的结构，就是用 JS 通过DOM 来改变。

  • 更新：更新该 DOM 文档的结构，更新 HTML 的内容
  • 遍历：遍历该 DOM 节点一下的节点
  • 添加：在该DOM节点下新增一个子节点，相当于动态增加了一个HTML节点
  • 删除：将该节点从HTML中删除，相当于删掉了该DOM节点的内容以及它包含的所有子节点。
